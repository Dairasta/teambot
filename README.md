# Team Task Bot — Telegram

Командний таск-менеджер у Telegram. Підтримує 4 учасники, категорії, пріоритети, статистику.

## Швидкий старт

### 1. Створити бота
1. Відкрий Telegram → знайди @BotFather
2. Надішли `/newbot`
3. Вкажи назву та username бота
4. Скопіюй токен (виглядає як `123456789:AABBcc...`)

### 2. Налаштувати код
Відкрий `bot.py`, знайди рядок:
```python
TOKEN = os.environ.get("BOT_TOKEN", "YOUR_TOKEN_HERE")
```
Для локального тесту замінити `YOUR_TOKEN_HERE` на свій токен.

### 3. Запустити локально (тест)
```bash
pip install -r requirements.txt
python bot.py
```

### 4. Деплой на Render.com (безкоштовно)

1. Завантаж файли на GitHub (новий репозиторій)
2. Зайди на https://render.com → Sign Up
3. New → Background Worker
4. Підключи свій GitHub репозиторій
5. У полі **Environment Variables** додай:
   - Key: `BOT_TOKEN`
   - Value: твій токен від BotFather
6. Build Command: `pip install -r requirements.txt`
7. Start Command: `python bot.py`
8. Натисни **Deploy**

## Команди бота

| Команда | Дія |
|---------|-----|
| `/start` | Запустити бота |
| `/menu` | Головне меню |
| `/cancel` | Скасувати дію |

## Можливості

- ➕ Додавати завдання з призначенням учаснику
- 📋 Переглядати всі завдання
- 👥 Фільтр по учаснику (Андрій, Оксана, Василь, Людмила)
- 📊 Фільтр по статусу (До виконання / В процесі / Перевірка / Виконано)
- ✅ Відмічати завдання виконаними
- 📈 Статистика команди з прогрес-барами

## Зміна учасників

У файлі `bot.py` знайди словник `MEMBERS` і відредагуй імена:

```python
MEMBERS = {
    "0": "Ім'я 1",
    "1": "Ім'я 2",
    "2": "Ім'я 3",
    "3": "Ім'я 4",
}
```
